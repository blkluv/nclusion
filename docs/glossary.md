# Localization Glossary: Haitian Creole and French

This glossary is the source of truth for all user-facing terms. The localization bundles in `apps/mobile/` must match these translations exactly. Haitian Creole is the default language; French is secondary.

## Notes

- Haitian Creole spellings follow standard orthography
- Terms should be reviewed by a native Creole speaker before production launch
- The glossary covers MVP scope only — additional terms should be added as features expand
- All strings in this glossary must appear in both language bundles shipped in the APK
